1、哔哩哔哩

官方文档：https://player.bilibili.com
使用示例：

bvid获取方式：

1)b站网页端：

例如：https://www.bilibili.com/video/**BV1ze411E7P8**/?vd_source=9a98e5106c620882fd7a935680bdcfd1
加粗部分即为bvid

2)b站手机app：

展开视频简介，简介上方的一串字符即为bvid

其余补充说明：

参数名 类型 必要 说明
aid number UGC 视频 ID。aid、bvid 选择其一即可
cid number UGC 视频 ID
bvid string 必要！ UGC 视频 ID。aid、bvid 选择其一即可
seasonId number OGV 视频 ID
poster boolean 展示封面
autoplay boolean 自动播放
muted boolean 静音
t number 跳转到媒体的初始时间点，单位：秒
danmaku boolean false 表示关闭弹幕，其他表示默认值
kind number 群组种类。非通用业务需要此参数
refer boolean 跳链时携带当前的 Referrer。用于合作方查询来自嵌入网站的跳转次数
p number 多 P 视频的集数。从 1 开始计数，若有 cid 参数，则此参数不生效

2、网易云音乐

官方文档：无

步骤：

1)打开网页版网易云，并进入要添加的音乐
2)点击”生成外链播放器”，进入后设置好参数
3)将生成好的html代码复制粘贴到网页源代码中位置
